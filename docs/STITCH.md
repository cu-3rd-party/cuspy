# Stitch Design Prompt

Design a comprehensive UI/UX for Cukiller, a high-stakes spy-themed game.

## Visual Direction
- **Theme**: Cyber-espionage, secret agency, dark terminal, "Top Secret" dossier.
- **Style**: Dark mode by default. Use a palette of deep blacks, charcoal greys, and high-contrast accent colors (e.g., neon green, amber, or electric blue) for critical data.
- **Aesthetic**: Mix of brutalist layout and polished high-tech interfaces. Incorporate elements like monospaced fonts, scanning lines, redacted text effects, and dossier-style paper textures for profiles.

## Required Screens & Features

### 1. Entry & Onboarding
- **Landing Page**: A high-impact introduction. Must include an "Enlistment" Call-to-Action (CTA) that feels like joining a secret organization. Clearly display basic game rules in a "Briefing" format.
- **Profile Edit / Agent Setup**: A form to set up the agent's identity (Name, Photo, "About" section). Should feel like filling out a government clearance form.

### 2. Core Gameplay
- **Kill Report Form**: A high-friction, detailed reporting interface. Include fields for evidence upload, location details, and a "confirm submission" step that feels weighty and final.
- **Target Intelligence**: A screen displaying the current target's dossier.
- **Tinder-like Target Selection**: A swipe-based interface for choosing the next target from a curated list of available candidates. Left swipe to pass, right swipe to select.
- **Multiple Targets Tracking**: A dashboard view allowing agents to track and manage multiple active targets simultaneously.
- **Non-Spy Missions**: A list of intelligence-gathering tasks (e.g., "Shadowing", "Dead Drop"). Include task descriptions, reward details, and completion status.

### 3. Progression & Rewards
- **Treasure Chests**:
    - **Chest Discovery**: A visual cue or map-based indicator for finding a chest.
    - **Chest Opening**: An immersive opening sequence/animation that reveals the loot (Gold, Perks, or Cosmetics).
    - **Chests/Loot Inventory**: A screen to view collected items and chest history.
- **Perks System**:
    - **Perk Store/Activation**: A menu to browse available perks (e.g., Invulnerability, Score Multipliers, Territory Expansion) and activate them.
    - **Active Perks Dashboard**: A clear way to see currently active perks and their remaining duration (countdown timers).

### 4. Administration
- **Admin Dashboard**: A central command center.
    - **Kill Moderation**: A queue of pending kills with side-by-side evidence comparison and Approve/Reject actions.
    - **User Management**: Ability to edit agent statuses or ban players.
    - **Global Events**: An interface to trigger "Smites" (e.g., making one agent a global target for all players).

## UX Goals
- Make every action feel like a tactical operation.
- Use "classified" animations (e.g., text decrypting) when revealing information.
- Ensure a seamless transition between the high-tension "Hunt" screens and the strategic "Progression" screens.
